# Changelog

All notable changes to this project will be documented in this file.

## [0.3.0] - 2026-05-15
### Changed
- Updated compatibility for Foundry VTT v14, Conan 2d20 system 2.5.0, and Argon Core 5.0.0.
- Updated README and release metadata for the v14 release line.
- Replaced deprecated `Dialog.confirm` usage with `DialogV2.confirm`.

### Fixed
- Fixed the Threaten action icon path for Argon Core v14 by using a local module icon.

## [0.2.0]
### Added
- Native Conan system launchers from Argon for melee and ranged attacks, Threaten, Skill Test, Recover, Treatment, Assist, and Exploit.
- Real NPC attack support using Conan NPC attack items.
- Ready and Brace workflow support.
- Declarative Sprint / Withdraw actions with chat messaging.
- English and Spanish release-ready strings.

## [0.1.0]
### Added
- Initial Conan-aware Argon provider, settings, resources, and baseline action buttons.
